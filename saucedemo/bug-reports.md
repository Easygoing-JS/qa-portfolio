BUG-001 - Возможность начать оформление заказа при пустой корзине.
Окружение: Chrome 145, Windows 10, www.saucedemo.com.
Предусловия: Пользователь авторизован как standard_user, в корзине нет товаров.
Шаги воспроизведения:
1) нажать на значок корзины
2) нажать Checkout
3) заполнить данные покупателя; First Name: John, Last Name: Doe, Zip/Postal Code: 123456.
4) нажать Continue
5) нажать Finish
Ожидаемый результат: Система не позволяет оформить заказ при пустой корзине и информирует пользователя о необходимости добавить товар.
Фактический результат: Система пропускает пользователя с пустой корзиной к прохождению всех этапов оформления заказа, вплоть до успешного оформления заказа.
Серьезность: Major
Приоритет: High

BUG-002 -  Поле First Name принимает цифры и спецсимволы при оформлении заказа.
Окружение: Chrome 145, Windows 10, www.saucedemo.com.
Предусловия: Пользователь авторизован как standard_user, в корзине есть хотя бы один товар.
Шаги воспроизведения:
1) нажать значок корзины.
2) нажать Checkout
3) в поле First Name ввести: "!@12 3$%"
4) в поле Last Name ввести "Doe"
5) в поле Zip/Postal Code ввести "123456"
6) нажать Continue
Ожидаемый результат:  Система показывает ошибку валидации и не позволяет перейти к следующему шагу.
Фактический результат:  Система принимает значение "!@12 3$%" и позволяет перейти к следующим шагам вплоть до завершения оформления заказа.
Серьезность: Major
Приоритет: High

BUG-003 - Невозможность удаления товара из корзины на странице со всеми товарами.
Окружение: Chrome 145, Windows 10, www.saucedemo.com.
Предусловия: Пользователь авторизован как problem_user.
Шаги воспроизведения: 
1) Добавить в корзину товар "Sauce Labs Fleece Jacket". 
2) Нажать кнопку Remove на карточке добавленного в корзину товара 
Ожидаемый результат: система удаляет товар из корзины.
Фактический результат: нажатие кнопки Remove на карточке товара не удаляет товар из корзины, но товар удаляется из корзины при нажатии Remove внутри самой корзины.
Серьезность: Major, 
Приоритет: Medium.
