## TC-001: Отображение верного количества единиц товара, находящегося в корзине, на иконке корзины при добавлении товаров на странице каталога товаров

**Предусловия:** Пользователь авторизован на сайте saucedemo.com, корзина пуста.

**Шаги:**
1. Перейти на страницу каталога товаров (https://www.saucedemo.com/inventory.html)
2. Добавить товар Sauce Labs Backpack в корзину, нажав кнопку Add to cart
3. Добавить товар Sauce Labs Bike Light в корзину, нажав кнопку Add to cart

**Ожидаемый результат:** На иконке корзины отображается "2".

## TC-006: Сохранение товаров в корзине после выхода и повторного входа в аккаунт

**Предусловия:** Пользователь на странице логина.

**Шаги:**
1. В поле Username ввести "standard_user"
2. В поле Password ввести "secret_sauce"
3. Удалить товары из корзины, если они там есть
4. Перейти на страницу каталога товаров (https://www.saucedemo.com/inventory.html)
5. Добавить товар Sauce Labs Backpack в корзину, нажав кнопку Add to cart
6. Добавить товар Sauce Labs Bike Light в корзину, нажав кнопку Add to cart
7. Добавить товар Sauce Labs Fleece Jacket в корзину, нажав кнопку Add to cart
8. Нажать иконку корзины
9. Выйти из аккаунта
10. В поле Username ввести "standard_user"
11. В поле Password ввести "secret_sauce"
12. Нажать иконку корзины

**Ожидаемый результат:** Система сохранила все добавленные товары в корзине, их порядок и кнопки действий.

**Тип:** Позитивный

## TC-007: Ввод цифр в поле First Name при оформлении заказа

**Предусловия:** Пользователь авторизован на сайте saucedemo.com; в корзине не менее одного товара; пользователь на странице ввода данных покупателя.

**Шаги:**
1. Ввести цифры в поле First Name: "123456"
2. Ввести буквы в поле Last Name: "Smith"
3. Ввести цифры в поле Zip/Postal Code: "12345"
4. Нажать Continue

**Ожидаемый результат:** Отображается сообщение об ошибке. Пользователь остаётся на странице ввода данных о покупателе.

**Тип:** Негативный
